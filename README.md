# CODSOFT – Machine Learning Internship

This repository contains my completed projects for the <a href="https://www.codsoft.in/">CodSoft</a> Machine Learning Internship. The projects focus on solving real-world Machine Learning and Natural Language Processing (NLP) problems using Python and popular ML libraries.

## Projects

### 1. Movie Genre Classification
Built a movie genre classification model using TF-IDF feature extraction with algorithms such as Naive Bayes, Logistic Regression, and Support Vector Machine (SVM).

### 2. Customer Churn Prediction
Developed a customer churn prediction model using customer demographic and behavioral data with Logistic Regression, Random Forest, and Gradient Boosting.

### 3. Spam SMS Detection
Created an SMS spam detection system using NLP techniques and classifiers like Naive Bayes, Logistic Regression, and SVM.

## Technologies Used

- Python
- pandas
- numpy
- scikit-learn
- matplotlib
- seaborn

Each project folder includes its own README with dataset details, setup instructions, and execution steps.
